# Story 21.2: Workshop Instructions Panel & WLED Setup Guide

**Epic:** Education Mode Workshop Readiness
**Type:** Brownfield Enhancement
**Estimated Effort:** 6-8 hours
**Status:** Draft
**Tags:** `Workshop`, `Education`, `WLED`, `Instructor Guide`
**Dependencies:** Story 21.1 (Boomwhacker Workshop Lesson structure)

---

## User Story

**As a** workshop instructor preparing for a Boomwhacker workshop,
**I want** a dedicated Workshop Instructions panel with WLED setup guidance,
**So that** I can quickly configure LED light strips (if available) and troubleshoot common issues without leaving the education interface.

- `EducationMode.tsx` - Existing lesson framework (Story 21.1 creates workshop lesson)

**Touch points:**

## Notes

**Design Decisions:**
1. **Panel collapsed by default** - Reduces visual clutter, instructor can expand when needed
2. **localStorage for state** - Simple persistence without server dependencies
3. **Radio buttons instead of tabs** - Clearer UX for two mutually exclusive options
4. **External links to WLED docs** - Authoritative source, always up-to-date
5. **Static content** - No device detection or auto-configuration (keeps scope minimal)

**Content Philosophy:**
- **Beginner-friendly** - Assume no prior WLED knowledge
- **Actionable** - Every instruction is a specific step, not conceptual
- **Troubleshooting-forward** - Anticipate common issues and provide solutions
